- Вилучення дубльованого коду (Extract Method / Remove Duplication) – виділення повторюваних фрагментів у окремі методи.

У роботі детально розглянуто принципи рефакторингу, основні методи його застосування та поняття “запахів коду”.

Аналіз “запаху” Duplicated Code показав, що уникнення дублювання дозволяє зберегти якість коду, зробити його більш гнучким та полегшити подальшу модифікацію і тестування.
