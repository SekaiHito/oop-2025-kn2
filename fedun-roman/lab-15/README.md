## Мета

Мета роботи – познайомитися з основними принципами та найбільш поширеними техніками рефакторингу програмного забезпечення.

## Шаблони

- Extract Method (Виділення методу) – винесення частини коду в окремий метод.
- Rename Variable (Перейменування змінної) – зрозумілі назви змінних.
- Replace Magic Number with Constant (Заміна "магічних чисел" константами).
- Inline Method (Вбудовування методу).
- Introduce Parameter Object (Введення об’єкта параметрів).

## Хід роботи

- Ознайомився з поняттям рефакторингу та його принципами.
- Вивчив основні техніки рефакторингу.
- Детально розглянув техніку Extract Method та реалізував приклад на Python.
- Ознайомився з поняттям "запахів коду" та описав приклад Duplicated Code.

## Висновок

У роботі розглянуто принципи рефакторингу, основні техніки та поняття "запахів коду".
На прикладі техніки Extract Method показано, як можна підвищити читабельність і підтримуваність програми.
Аналіз "запаху" Duplicated Code довів важливість уникнення дублювання для збереження якості та гнучкості коду.
