Про виконання лабораторної роботи №11 з дисципліни "Об'єктно-орієнтоване програмування" на тему: "Твірні шаблони проєктування"

Виконав: студент групи КН-32СП Цар Володимир

Прийняв: ст. викл. Н. Заплатинський

Львів 2025
Мета: Вивчення особливостей та принципів застосування групи твірних шаблонів проєктування.

Хід роботи
1. Теоретичний опис твірної групи шаблонів. Твірні патерни (Creational Patterns) — це фундаментальна категорія шаблонів проєктування, що відповідає за механізми створення об'єктів. Головне завдання цих шаблонів — відокремити процес ініціалізації сутності від її безпосереднього використання. Це забезпечує архітектурну гнучкість, дозволяючи масштабувати систему та змінювати типи створюваних об'єктів без необхідності втручання в основну бізнес-логіку програми.

2. Реалізація фабричного методу. У ході роботи було проаналізовано патерн "Фабричний метод" (Factory Method). Розроблено програмну реалізацію даного шаблону мовою Python з використанням абстрактних класів, що демонструє принцип поліморфізму та інкапсуляції створення об'єктів.

##Висновки
Під час виконання лабораторної роботи було розглянуто групу твірних шаблонів проєктування. Зокрема, детально вивчено принцип роботи "Фабричного методу", реалізовано його програмний код та побудовано відповідну UML-діаграму класів для візуалізації архітектури рішення.
