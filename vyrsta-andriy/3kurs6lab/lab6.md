### 1. Ознайомлення з видами тестів та пірамідою тестування

Було опрацьовано такі види тестів: - **Unit-тести** --- тестують окремі
функції або класи. - **Integration-тести** --- перевіряють взаємодію
компонентів. - **System-тести** --- тестують роботу всієї системи. -
**Acceptance-тести** --- перевіряють відповідність вимогам замовника.

### 2. Створення Python-програми для тестування

Було розроблено невелику програму з двома функціями: `add_numbers` та
`is_even`.

### 3. Написання unit-тестів

За допомогою `unittest` були створені тести для перевірки коректності
роботи функцій.

------------------------------------------------------------------------

## **Код програми (lab16.py)**

``` python
def add_numbers(a, b):
    return a + b

def is_even(n):
    return n % 2 == 0
```

------------------------------------------------------------------------

## **Unit-тести**

``` python
import unittest

class TestFunctions(unittest.TestCase):

    def test_add_numbers(self):
        self.assertEqual(add_numbers(2, 3), 5)
        self.assertEqual(add_numbers(-1, 1), 0)
        self.assertEqual(add_numbers(0, 0), 0)
        self.assertNotEqual(add_numbers(10, 5), 20)

    def test_is_even(self):
        self.assertTrue(is_even(4))
        self.assertTrue(is_even(0))
        self.assertFalse(is_even(5))
        self.assertFalse(is_even(101))

if __name__ == "__main__":
    unittest.main()
```
